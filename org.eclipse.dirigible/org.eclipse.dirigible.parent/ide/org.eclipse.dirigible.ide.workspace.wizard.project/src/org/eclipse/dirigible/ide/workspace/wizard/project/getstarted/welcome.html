The Get Started Project will contain the main artefacts needed to run an end to end service.
It will include:
  • a database table to persist the data;
  • a RESTfull JavaScript server-side service;
  • and user interface in HTML5

More advanced technics such as creation of Flows and Jobs, Wiki pages, Extensions and Tests can be found at http://help.dirigible.io and http://samples.dirigible.io
